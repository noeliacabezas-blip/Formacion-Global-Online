Curso de Excel Avanzado

Con Power Query y Power Pivot

  • Categoría: Tecnología
  • Nivel: Avanzado
  • Requisitos previos: Conocimientos medios de Excel
  • Duración: 400 horas
  • Profesor responsable: José Álvarez Álvarez

Descripción:

Deja de usar Excel solo como una hoja de cálculo. Con este curso de excel avanzado online aprenderás a dominar las herramientas más potentes de Microsoft para llevar tus datos al siguiente nivel.

Incluye:

  • Power Query para limpiar y transformar información de forma automática.
  • Power Pivot para modelar grandes volúmenes de datos con rapidez.
  • Power BI para crear paneles interactivos y visualizaciones que facilitan la toma de decisiones.
